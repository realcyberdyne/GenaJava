# How use Generator

### 1-Install JDK 17
```
sudo apt install openjdk-17-jdk
```

### 2-Install Maven On System
```
sudo apt install maven
```

### 3-Clone Project to system
```
git clone https://github.com/realcyberdyne/Generator.git
```

### 4-Run Project
```
mvn exec:java
```
